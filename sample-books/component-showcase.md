## Chapter 1. 컴포넌트 쇼케이스

이 원고는 엔진이 지원하는 추가 블록들을 한자리에서 보여 주는 검수/테스트용입니다.
판매용 책에는 포함하지 않습니다.

도구를 고를 때는 무료와 유료를 비교해 보세요.

:::compare-card
columns: 기능, 무료 도구, Pro 도구
highlight: Pro 도구
- 기본 편집, ○, ✓
- 고급 자막, ✕, ✓
- 워터마크 제거, ✕, ✓
- 비용, 0원, 월 구독
:::

:::alert
variant: success
첫 영상을 끝까지 완성했습니다. 가장 큰 고비를 넘겼습니다.
:::

:::alert
variant: warning
효과를 한 번에 너무 많이 넣지 마세요.
:::

제작은 큰 흐름으로 보면 세 단계입니다.

:::process
icon: ✍
title: 기획
desc: 주제와 대본을 정합니다.

icon: 🎬
title: 제작
desc: 촬영·편집을 합니다.

icon: 🚀
title: 발행
desc: 업로드하고 공유합니다.
:::

독자 평가:

:::rating
value: 4.5
max: 5
label: 입문자 추천도
:::

이 책의 키워드 묶음입니다.

:::tags
- AI 영상
- 초보자
- 무료 도구
- 숏폼
:::

:::chips
대본, 편집, 자막, BGM, 썸네일
:::

작업 폴더는 이렇게 정리하면 좋습니다.

:::tree
- 영상 프로젝트
  - 소스
    - 이미지
    - 음원
  - 편집본
- 완성본
:::

:::pagination
current: 7
total: 7
:::

아직 업로드한 영상이 없을 때:

:::empty
icon: 📭
title: 아직 영상이 없습니다
desc: 첫 영상을 업로드하면 이 자리에 표시됩니다.
:::

자료를 찾을 때:

:::search
placeholder: 편집 프로그램 단축키 검색
:::

:::tooltip
label: 알아두기
text: 숏폼은 도입 3초가 가장 중요합니다.
:::

:::popover
title: 추가 안내
text: 자막 폰트는 1~2종으로 고정하면 채널 일관성이 생깁니다.
:::

:::modal
title: 업로드 전 점검
text: 오타·소리 크기·첫 3초를 마지막으로 확인하세요.
:::

:::drawer
title: 빠른 메뉴
text: 내보내기 · 썸네일 만들기 · 설명 작성 · 예약 업로드
:::

자료를 불러오는 동안 표시되는 자리입니다.

:::skeleton
lines: 3
:::
