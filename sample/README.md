# sample/ — 판매 자료 참고

판매용 산출물은 이제 `npm run build` 가 **`output/` 에 자동 생성**합니다.

| 산출물 | 위치 | 설명 |
| --- | --- | --- |
| 크몽 미리보기 PDF | `output/kmong-preview.pdf` | 구매자용 샘플 PDF |
| 상세페이지 이미지 | `output/detail-images/` | 대표/홍보/챕터별 PNG 묶음 |

이 `sample/` 폴더는 수동 큐레이션/보관용 참고 공간입니다.
필요하면 `output/` 의 산출물을 골라 이곳에 정리해 두고 쓰세요.

```text
sample/
  detail-images/   ← 골라 둔 상세 이미지 보관(선택)
```
